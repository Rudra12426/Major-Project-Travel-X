## вљ™пёЏ Getting Started

### Prerequisites

- [XAMPP](https://www.apachefriends.org/) or any local PHP server
- Python 3.x (for the AI assistant)
- A modern web browser

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Rudra12426/Major-Project-Travel-X.git
   ```

2. **Move to your server's root directory**
   ```bash
   # For XAMPP (Windows)
   Move the folder to: C:/xampp/htdocs/major/
   ```

3. **Start Apache** from the XAMPP Control Panel

4. **Open in browser**
   ```
   http://localhost/major/index.php
   ```

5. *(Optional)* **Run the AI Assistant**
   ```bash
   python app.py
   ```
